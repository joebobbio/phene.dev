Introduction

The matrix <=> Discord bridge is a method to "bridge" or link two channels from different services together.


The way this works is that a bot in both the matrix room and Discord server listens for messages and pipes them to both platforms.
This allows for seamless communication between both platforms.


Setup

First, you must create an UNENCRYPTED room on matrix.

Then, you must add the matrix bot to your room - invite @_discord_bridging:matrix.phene.dev to the room.

Lastly, add the Discord bot to the server you wish to bridge to.
